dMVC is a library for a single-page Web application development.

Hardware requirements

Framework transfers the entire application logic on the server work. This approach reduces browser hardware requirements.

Complexity

It combines in some single system all the components of the MVC-pattern. This allows to reduce the complexity of the overall system.

Visibility

All of the logic can be transferred to the server. This will help hide the implementation from the client.

dMVC is open source. It's hosted, developed, and maintained on GitHub
